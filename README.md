# Newsletter Sign-Up Form with Success Message

## Overview

This project is a recreation of the Newsletter Sign-Up Form challenge. The goal was to replicate the supplied design as closely as possible using HTML, CSS, and JavaScript.

The application allows users to subscribe to a newsletter by entering an email address. The form validates the email input, displays an error message for invalid entries, and redirects successful submissions to a confirmation page.

---

## Features

* Responsive newsletter sign-up form
* Email validation using JavaScript
* Error state for invalid email addresses
* Success confirmation page
* Dynamic display of the submitted email address
* Hover and active button states
* Clean separation of HTML, CSS, and JavaScript

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)


## Functionality

### Newsletter Form

Users can:

* Enter an email address
* Submit the form
* Receive validation feedback if the email is invalid

### Error Handling

If an invalid email is entered:

* An error message is displayed
* The input field receives an error style
* The user remains on the sign-up page

### Success Page

If a valid email is entered:

* The user is redirected to the success page
* The submitted email is displayed in the confirmation message
* The user can return to the sign-up page using the Dismiss Message button
